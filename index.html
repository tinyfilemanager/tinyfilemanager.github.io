Tiny File Manager

Web based File Manager in PHP, Manage your files efficiently and easily with Tiny File Manager and it is a simple, fast and small file manager with a single file.

Open Source, light and extremely simple, It is a single executable file with no dependencies. Just download it and add it to your PATH

Create, edit, copy, move, download your files easily, everywhere, every time. Use it as your personal cloud.

Ajax Upload, Ability to drag & drop, multiple files upload and upload using url with file extensions filter

Multiple users are supported, each one with their own folder, permissions and access roles.

Search through your files and find the one instantly. Filter by type, by extension...

Ability to compress, extract files. Create archives with zip or unzip and decompress files online

Responsive Mobile Layout

It is a powerful-lightweight fully responsive based on Bootstrap 4, you know that it will look great on any device, whether it's a phone, tablet, or desktop the page will behave responsively!

Multi Upload

Advanced AJAX multiple file upload with progress bar. Drag & Drop from your local drive directly to browser's window to start upload. Flash free. we used DropzoneJS and Upload files using URL.

Create & Edit Files

A browser based code editor. It is specialized for editing code, and comes with a number of language modes. IDE have syntax highlighting for over 150+ languages, Over 35+ themes with your favorite programming style.
